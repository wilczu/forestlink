Safety

You can report any website which you think is causing any harm for users and we will block it. By blocking that page, we will prevent users from adding this site to their desktops. All of the reports are checked by our staff. You can also view the status of the domain name which you reported on the report site.

Control

Set custom background of the website tile and colour of the text. There is no strict rule about the URL of your site it can be a numeric IP address of anything inside of your network or just a regular domain name. You can edit or remove your page later on.

Icons

By using then you would be able to find the page which you're looking for much quicker by looking at the logo of the page. You don't need to browse all of the icons to find the right one, just type the name of the platform and you will see the right icon for your page.
